# 基于MCP与QWEN大模型的自然语言MySQL数据库查询系统

## 环境

- NodeJS 目前的stable版本22.13.1，所以node@^22.13.1是允许的
- pnpm@^9.5.0

## V2 - 快速开始

- 确认环境后，`pnpm i`安装依赖
- 复制根目录的`.env.example`文件为`.env`文件，并填写相关内容
- `pnpm start`启动应用
